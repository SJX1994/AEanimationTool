前言：
游戏通常是图形密集型活动，您的计算机需要尽可能地发挥最佳性能。根据您正在玩的游戏的需求，GPU 使用率应该在 70% 到 100% 之间。GPU 使用率的下降会导致性能下降，或者专家称之为游戏中的每秒帧数 (FPS)。
Q：在跑游戏的时候GPU使用率多少是正常的？
A：高要求的游戏可以让 GPU 几乎 100% 运行，这是正常的。高 GPU 使用率意味着游戏使用了 GPU 的所有可用 FPS 或性能。实际上，如果图形密集型游戏的 GPU 使用率不高，您应该担心。除非您的计算机处于空闲状态，否则游戏时 GPU 使用率较高是完全正常的。您的显卡经过精心设计，可在数年内以近乎 100% 的速度充分利用，尤其适用于游戏等 GPU 密集型任务。因此，预计会有高 GPU 使用率。在玩大多数高图形游戏时，预计会达到 90% 到 95% 的 GPU 使用率。 如果您在游戏中保持 80% 并达到 55 到 50 FPS，这可能是 CPU 速度瓶颈问题的征兆。如果您在游戏中的 FPS 很高，那也没关系，因为这也表明游戏要求很高，此时，GPU 的使用率应该是最大的。 如果您的图形处理单元的 温度不超过 185 华氏度（85 摄氏度） ，那么玩游戏时GPU 使用率达到 100% 是正常的。如果温度太高（85+ 摄氏度），您的性能可能会随着时间的推移而降低。

Q：GPU 使用率高、温度高、FPS 低 之间的关系？
A：有些游戏旨在充分利用您的 GPU，这是一件好事。 如果您的 GPU 使用率高、温度高且性能低，这将是一个坏消息。只要性能和温度可接受（高于 55FPS 和低于 185 华氏度），高 GPU 使用率是正常的。如果您的 GPU 使用率为 100% 并且在玩某些游戏时温度很高，您可能会遇到输入延迟。您可以通过限制 FPS 来降低 GPU 使用率。 将 GPU 降低到某个水平，例如 95%，可以帮助减少延迟、降低温度并增加延迟。  

Q：GPU 使用率高，CPU 使用率低——这正常吗？
A：是的，这完全正常。这意味着您可以从 GPU 获得最佳的游戏内性能，并且您的 CPU 在此过程中不会受到伤害。 高 GPU 和低 CPU 使用率是您在玩游戏时应该期待的。在执行此类图形密集型任务时， 您的 GPU 应该是系统的瓶颈，而不是 CPU。 

结论：
70% 到 100% 的 GPU 使用率对于游戏来说是正常的。范围取决于您正在玩的游戏类型。有些游戏不像其他游戏那样需要大量图形，在这种情况下，大约 70% 的 GPU 使用率是可以接受的。 
相反，大多数游戏可以让您的 GPU 使用率达到 90% 甚至高达 100%。如果您在游戏中的 FPS 和温度分别高于 55 华氏度和低于 185 华氏度，则高 GPU 是正常的。我们还了解到，高 GPU 使用率和高温可能会导致延迟问题。您可以通过限制 FPS 将您的 GPU 使用率提高到一定水平，以帮助解决此输入延迟问题。
