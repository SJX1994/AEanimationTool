            数学：
                  未知数/自由度 与 条件 要一样多：

                        例子：
                              1元1次方程：
                                    一个条件：
                                          2x - 1 = 45
                                    一个未知数：
                                          x
                              1元2次方程：
                                    一个条件：
                                    pow(x,2)+ 2x - 1 = 0
                                    一个未知数：
                                          x 
                              2元1次方程：
                                    两个条件：
                                    两个未知数
                              3元1次方程：
                                    三个条件
                                    三个未知数
                                    
                              特例：
                                    一个条件 解多个未知数：
                                          pow(x,2) + pow(y,2) = 0;
                                          x = 0;
                                          y = 0;
                                          这是边缘情况，根据圆的公式，pow(x,2) + pow(y,2) = pow(r,2) 当R = 0，圆被缩小成了点
                        理解：
                              空间维数图形化方程：
                                    未知数相当于坐标轴
                                    条件相当于线
                        结论：
                              数学结论：
                                    每多一个未知数，空间增加一个维度
                                    每多一个条件，空间减少一个维度
                                    当空间减少到0维度，便是若干点
                                    点就是解集
                              物理结论：
                                    每多一个未知数，“ 自由度 ” 增加 “ 1 ”
                                    每多一个条件， “ 自由度 ” 减少 “ 1 ”
                                    当自由度被降低至 “ 0 ”
                                    完全不自由 定死形态
                        我的思考：
                              这还可以从 游戏设计的角度 给出越多能力 游戏自由度 越高，设定越多规则 游戏自由度 越低，当完全一个能力对应一个自由度时，游戏就变成线性叙事了，就和电影差不多了
