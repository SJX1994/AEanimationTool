回收的游戏：
      公式：
            目标：

                  获取单品类最合理的收费价格

                  已知：

                        收购单个的价格：x 

                        商品出售单个的价格（变量）：z

                        持有商品数量：n

                        x = n * (x + x*0.1)

                        存货控制：np = 0.1

                        成本利润率 = cp

                        
                  求：
            
                        x 与 z n np cp 之间的关系

                  
                  解：
                        根据成本利润公式：

                              利润率 = 售价-进价/售价

                              cp = z -x / z
                              cp*z = z -x
                              z-x = cp*z
                              -x = cp*z -z
                              x = z-cp*z -(x*n*np)

                        cp = (z - (x-x*n*np)) / z
                       z*cp = z - (x-x*n*np)
                  
                              z - z*cp = x-x*n*np
                              (z - z*cp) / (1-n*np) = x
