节点:

      

      Is Front Face Node （ 渲染的是正面或背面 ）：

            如果当前渲染正面，则返回true；如果渲染背面，则返回false。

            注意事项： 
                  只能作用与双面渲染，多用于Branch Node


      Branch Node（ 分支节点/条件节点 ）：

            为着色器提供动态分支

            输入：



            输出：

      Position( 位置节点 ):

            提供对网格顶点或片段的“空间位置”的访问

            模式：
                  对象
                  视图
                  世界
                  切线
                  世界绝对

            输出：
                  v3


      Screen Position( 屏幕坐标 ):

            提供对网格顶点或片段的“屏幕位置”的访问

            模式：
                  Default：
                        返回屏幕位置。= 屏幕位置/剪辑空间 W 分量
                        clip space position（视口空间，剪辑空间）：
                              个参考系，一个坐标系，即用于指定点和向量的特定的原点选择和三个轴的集合。
                              发生在光栅化之前，顶点处理器之后
                              其他：
                                    NDC是设备坐标归一化
                  Raw：
                        返回屏幕位置
                  Center：
                        返回屏幕位置偏移量，以便位置float2(0,0)位于屏幕的中心。
                  Tiled：
                        返回屏幕位置偏移量，以便位置float2(0,0)位于屏幕的中心，并使用进行平铺frac（分数）
      
      Scene Depth Node（场景深度节点）：

            使用输入UV提供对当前Camera深度缓冲区的访问，该输入应为归一化的屏幕坐标。

            注意事项：
                  深度缓冲区访问要求：
                        1“渲染管线”上启用深度缓冲区
                        2只能作用于片元着色器
                        3不同的渲染管线会返回不同的结果
            输入：
                  uv V4
            输出：
                  V1
            模式：
                  Linear01:
                        线性深度值介于0和1之间
                  Raw：
                        原始深度值
                  Eye:
                        深度转换为眼空间单位

      Tiling And Offset Node(平铺和偏移节点):

            通过输入“平铺”和“偏移”分别平铺和偏移输入UV的值。通常用于细节贴图和随时间推移滚动纹理。
            输入：
                  uv
                  Tiling
                  Offset
            输出：
                  v2
      
      Scene Color Node（场景深度节点）：  

            使用输入UV提供对当前Camera的颜色缓冲区的访问，该输入应为归一化的屏幕坐标。

            注意事项：
                  1不同的渲染管线会返回不同的结果
                  2该纹理的内容仅适用于透明对象。将主节点的“材料选项”面板上的“表面类型”下拉列表设置为“透明”，以从该节点接收正确的值。
                  3只能作用于片元着色器

            输入：
                  uv
            输出：
                  v3

      step (步骤)：

            只输出1或0
