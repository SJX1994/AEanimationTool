Adia_UVSetEditorPlus使用方法：

目标：
	按照客户提交要求，删除默认UVSet（默认情况下无法删除）
步骤：
	1.安装插件，将mel脚本拖入或者以记事本打开复制（Window>UI Element>Command Line）
	2.打开插件（Window>UI Element>Shelf>Adia_Tools>UVSetEditorPlus），输入需要置顶的UV Set（确保每个物体UV Set Editor都有填入的名字）
	3.选择单个或者多个物体，打开UV>UV Set Editor，复制需要置顶的UVSet名字
	4.UVSet名字输入插件，运行（结果：原先第一个默认UVSet被 输入名字的UVSet 顶替）
